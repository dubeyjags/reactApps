# React Apps

Some work with React which includes building interactive UIs and making API calls with FreeAPI.

---

## Authentications

Frontend with React and API calls with FreeAPI.

**Features:**
- Sign Up — register a new user account with form validation
- Sign In — authenticate existing users with error handling
- Profile — view logged-in user's profile details
- Local Storage — persist login sessions across page refreshes

**Tech used:** React, FreeAPI, LocalStorage

---

## Videos

Frontend with React and API calls with FreeAPI to display YouTube-style video content.

**Features:**
- Fetch live YouTube video data from FreeAPI
- Display video thumbnails, titles, and metadata
- Custom card design for each video item

**Tech used:** React, FreeAPI, Card Components
